Sistema de Gestión de Calentadores Solares

Usuarios del Sistema

  • Administrador: Puede gestionar usuarios, acceder a panel administrativo, visualizar y exportar logs.
  • Cliente: Puede ingresar y visualizar el contenido protegido.

Funciones Principales

  • Inicio de sesión con autenticación y sesiones.
  • Panel de administración con:
    • Alta, edición y eliminación de usuarios.
    • Control de roles: administrador y cliente.
  • Registro de actividad:
    • Cada acción realizada por un administrador se registra en una tabla logs.
    • Incluye fecha, usuario y descripción de la acción.
  • Exportación de logs:
    • Descarga de historial de actividades en formato Excel (.xls).
  • Protección de páginas por rol (solo admin accede al panel).
  • Base de datos MySQL lista para importar.

Archivos Incluidos

  • login.php: Página de inicio de sesión.
  • calentadores.php: Página principal del sistema.
  • config.php: Conexión a base de datos mediante variables de entorno.
  • auth_content.php: Lógica de sesión.
  • admin_panel.php: Panel de administración completo.
  • export_logs.php: Exportador de logs a Excel.
  • db/calentadores.sql: Script de base de datos MySQL.
